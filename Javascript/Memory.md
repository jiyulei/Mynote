### Stack vs Heap

| Aspect            | Stack                               | Heap                       |
| ----------------- | ----------------------------------- | -------------------------- |
| Data stored       | Primitive types, References to heap | Arrays, Objects, functions |
| Size              | Fixed and small                     | Dynamic and large          |
| Access speed      | Faster                              | Slower                     |
| Memory management | Automatically (LIFO)                | Garbage collection         |
